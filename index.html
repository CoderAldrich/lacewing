What is Lacewing?

Lacewing (or more properly liblacewing) is a cross-platform, high-level networking library.

It aims to provide a set of straightforward classes for socket communication in C/C++, with a focus on scalability and platform specific optimisations (eg. IOCP, epoll and kqueue).

Lacewing is open source under a BSD-style license.

Where can I get it?

The current version is 0.4.0 (zip - tarball).

Other versions are available over at GitHub. The latest commit for each released version is labeled with a tag, or you can just take the HEAD (which should always at least compile, but may not match the documentation).
