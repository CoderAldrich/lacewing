liblacewing is a cross-platform, high-level networking
library for C, C++ and Javascript.

Features :-
  • Runs natively on Windows, BSD, Linux and Mac OS X
  • Uses IOCP, kqueue or epoll for maximum scalability
  • Released under the Simplified BSD License

The latest release is 0.2.3 (31st August 2011)

Binaries

Binaries are currently only provided for Windows.

• lacewing-0.2.3-win32.zip

Source

The complete Lacewing source is always available at GitHub.
