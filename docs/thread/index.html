lacewing::thread

Description

thread provides a simple, cross-platform facility for starting and joining a named thread.

The constructor for thread takes two parameters:

  • A pointer to the thread procedure
  • A user-defined name for the thread

The thread procedure may take a single pointer-sized parameter (as will be passed to start) and return a 32-bit integer (returned by join).

When working with threads, remember that the majority of liblacewing is not thread-safe, with the exception of pump::post and similar.
