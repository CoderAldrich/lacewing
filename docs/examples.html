Examples

Examples for liblacewing can be found in the examples directory (git).

Currently they're all webserver examples, but they should at least give you a feel for how the liblacewing API works (and determine if you managed to build it correctly) :-

  • hello_world - Outputs "Hello world" and lacewing::version in response to any request.
  • send_file - Demonstrates the use of WriteFile to display its own source code.
  • ajax - Simple long-poll AJAX example using jQuery.

More examples will be added here over time. If you have something you'd like included, feel free to issue a pull request to have it added to the examples folder.
