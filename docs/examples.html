Examples

Examples for liblacewing can be found in the examples folder (git).

Currently they're all Webserver examples, but they should at least give you a feel for how the Lacewing API works (and determine if you managed to build it correctly) :-

  • hello_world - Outputs "Hello world" and Lacewing::Version in response to any request.
  • send_file - Demonstrates the use of SendFile to send its own source code.
  • ajax - Long-poll AJAX example using jQuery.

More examples will be added here over time. If you have something you'd like included, feel free to issue a pull request to have it added to the examples folder.
