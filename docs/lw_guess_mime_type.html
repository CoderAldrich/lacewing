lw_guess_mime_type

C
void lw_guess_mime_type (const char * name)

Description

Guesses an appropriate MIME type for a filename or file extension (ie. "text/css" for "css")

If the file extension is not known, "application/octet-stream" is returned.

Parameters

Name

A filename or file extension. If the name contains any . characters, the rightmost and everything before it will be ignored (thus allowing complete filenames without the application having to perform any additional parsing).
