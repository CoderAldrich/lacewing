Next (RelayClient::ChannelListing)

C++
Lacewing::RelayClient::ChannelListing * Lacewing::RelayClient::ChannelListing::Next ()
C
lw_relayclient_channel_listing * lw_relayclient_channel_listing_next
    (lw_relayclient_channel_listing *)
JS
The RelayClient class is not currently supported in Javascript

Description

Returns the next ChannelListing, or 0 if this ChannelListing is the last.

Combined with RelayClient::FirstChannelListing, this can be used to iterate through the entire channel list. The order of the channels is unspecified.
