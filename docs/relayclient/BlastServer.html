BlastServer (RelayClient)

C++
void Lacewing::RelayClient::BlastServer
    (int Subchannel, const char * Data, int Size = -1, int Variant = 0)
C
void lw_relayclient_blast_server
    (lw_relayclient *, long subchannel, const char * data, long size, long variant)
JS
The RelayClient class is not currently supported in Javascript

Description

Blasts data to the server, with a subchannel and optional variant identifier.

Parameters

Subchannel

The subchannel to blast the data on (0 - 255)

Data

A pointer to the data to blast

Size [optional]

The size of the data in bytes, or -1 if the data is a null-terminated string

Variant [optional]

A user-defined integer ranging from 0 - 15 to identify the type of the data.
