Join (RelayClient)

C++
void Lacewing::RelayClient::Join (const char * Channel, bool Hidden = false, bool AutoClose = false)
C
void lw_relayclient_join (lw_relayclient *, const char * channel)
C
void lw_relayclient_join_ex (lw_relayclient *, const char * channel, lw_bool hidden, lw_bool auto_close)
JS
The RelayClient class is not currently supported in Javascript

Description

Attempts to join a channel. Depending on whether the join was successful, onJoin or onJoinDenied will be triggered.

The protocol requires that a name has been set successfully (see Name) before it is possible to join a channel. This means Join should not be called before onNameSet triggers.

Parameters

Channel

The name of the channel to join.

Hidden [optional]

This parameter only applies if this RelayClient is creating the channel (that is, the channel didn't already exist), and thus will be the channel master.

Set to true to have the channel hidden from the channel list, or false (default) to have it show up as normal.

AutoClose [optional]

Like Hidden, this parameter only applies when creating the channel.

Set to true to have the entire channel close when this RelayClient (the channel master) leaves, or false (default) to have it stay open without a channel master.
