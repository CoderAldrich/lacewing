Next (RelayClient::Channel::Peer)

C++
Lacewing::RelayClient::Channel::Peer * Lacewing::RelayClient::Channel::Peer::Next ()
C
lw_relayclient_peer * lw_relayclient_peer_next
    (lw_relayclient_peer *)
JS
The RelayClient class is not currently supported in Javascript

Description

Returns the next Peer in the Channel, or 0 if this Peer is the last.

Combined with Channel::FirstPeer, this can be used to iterate through every Peer on the Channel. The order of the peers is unspecified.
