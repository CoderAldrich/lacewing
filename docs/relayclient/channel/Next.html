Next (RelayClient::Channel)

C++
Lacewing::RelayClient::Channel * Lacewing::RelayClient::Channel::Next ()
C
lw_relayclient_channel * lw_relayclient_channel_next
    (lw_relayclient_channel *)
JS
The RelayClient class is not currently supported in Javascript

Description

Returns the next Channel this RelayClient has already joined, or 0 if this Channel is the last.

Combined with RelayClient::FirstChannel, this can be used to iterate through the entire list of channels. The order of the channels is unspecified.
