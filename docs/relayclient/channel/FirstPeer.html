FirstPeer (RelayClient::Channel)

C++
Lacewing::RelayClient::Channel::Peer * Lacewing::RelayClient::Channel::FirstPeer ()
C
lw_relayclient_peer * lw_relayclient_channel_first_peer (lw_relayclient_channel *)
JS
The RelayClient class is not currently supported in Javascript

Description

Returns a pointer to the first Peer on this Channel, or 0 if there are no peers.

This can be used to iterate through every peer on the channel using Peer::Next
