LastModified

C++
lw_i64 Lacewing::LastModified (const char * Filename)
C
lw_i64 lw_last_modified (const char * filename)
JS
Lacewing.lastModified (filename)

Description

Returns the time a file was last modified, or 0 if the file does not exist.

The time is returned in UNIX time (number of seconds since January 1st, 1970)

Parameters

Filename

Any absolute or relative filename.
