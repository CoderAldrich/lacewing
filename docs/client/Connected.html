connected (client)

C++
bool lacewing::client::connected ()
C
lw_bool lw_client_connected (lw_client)

Description

Returns true if the client is currently connected to a server, false otherwise.

This will return false right up until the connection is properly established. To determine if the client is in the process of connecting to a server, see connecting.
