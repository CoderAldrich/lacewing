connect (client)

C++
void lacewing::client::connect (const char * host, int port)
C++
void lacewing::client::connect (lacewing::address)
C
void lw_client_connect (lw_client, const char * host, long port)
C
void lw_client_connect_addr (lw_client, lw_addr)

Description

Connects to a server. When the connection is established, on_connect will trigger.

Parameters

Host

The hostname or IP address to connect to.

Port

The port to connect on. If the host string contains a port (e.g. "example.com:80"), it will override this parameter.

OR address

An address object specifying the host to connect to.
