CheapBuffering (Client)

C++
bool Lacewing::Client::CheapBuffering ()
C
lw_bool lw_client_cheap_buffering (lw_client *)
JS
The Client class is not currently supported in Javascript

Description

Returns true if buffering is considered "cheap" for this client, or false otherwise.

If buffering is considered "cheap", it means the platform provides kernel-level buffering (ie. TCP_NOPUSH or TCP_CORK). Otherwise, Lacewing will perform "expensive" buffering by copying the data in userland.

Buffering is notably not cheap on Windows (where neither TCP_NOPUSH nor TCP_CORK exist) and Mac OS X (where TCP_NOPUSH does not flush the data immediately).
