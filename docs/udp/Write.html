Write (UDP)

C++
void Lacewing::UDP::Write (Lacewing::Address &, const char * Data, size_t Size)
C
void lw_udp_write (lw_udp *, lw_addr *, const char * data, size_t size)

Description

Sends a datagram to a remote host specified by an Address object.

Parameters

Address

An Address object representing the address to send the datagram to.

Data

A pointer to the data to send.

Size [optional]

The size of the data in bytes, or -1 if the data is a null-terminated string.
