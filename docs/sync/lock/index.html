(No classes, methods or handlers)

Lacewing::Sync::Lock

Description

Create an instance of a Lock object to acquire exclusive access to the resource protected by a Sync.

Only one Lock object may be in scope for each Sync object at any given time. The Lock constructor will block if the given Sync is currently locked by another thread.
