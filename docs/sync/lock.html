lock (sync)

C++
{ lacewing::sync::lock lock (some_sync);
    /* we now have exclusive access to the resource protected by some_sync */
}
C
void lw_sync_lock (lw_sync);

Description

Call #lw_sync_lock# (C) or create an instance of a lock (C++) to acquire exclusive access to the resource protected by a sync.

#lw_sync_lock# or the lock constructor will block if the given sync is currently locked by another thread.
