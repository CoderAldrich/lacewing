Lacewing::Sync

Description

Sync provides a cross-platform synchronization object, similar to a CRITICAL_SECTION on Windows, or a recursive pthread mutex.

It enables the protection of sections of code or variables from simultaneous access by different threads. For a more complicated synchronization object designed for "multiple readers, one writer", see #SpinSync#.
