enable_manual_finish (webserver)

C++
void lacewing::webserver::enable_manual_request_finish ()
C
void lw_ws_enable_manual_request_finish (lw_ws)

Description

After this function has been called, the response to a request will not be sent immediately at the end of the request hook. The change is global to all requests and permanent.

If this is enabled, the application must call request::finish at some point for every request object passed to a request hook.

This is particularly useful for implementing "long-poll" AJAX, because it means the response can be sent long after the request.
