onUploadDone (Webserver)

C++
void onUploadDone (Lacewing::Webserver &Webserver, Lacewing::Webserver::Request &Request,
                       Lacewing::Webserver::Upload &Upload)
{
}

Webserver.onUploadDone (onUploadDone); /* to register the handler */
C
void on_upload_done (lw_ws * webserver, lw_stream * request, lw_ws_upload * upload)
{
}

lw_ws_onuploaddone (webserver, on_upload_done); /* to register the handler */

Description

The onUploadDone handler is called to indicate that no more calls to onUploadChunk will be made for this Upload.

When all Upload objects passed to onUploadStart for this request have also had onUploadDone called, the onUploadPost handler will be called to indicate a complete upload post request.

This handler will never be called if Upload::SetAutoSave was called in onUploadStart.
