IdleTimeout (Webserver)

C++
int Lacewing::Webserver::IdleTimeout ()
C++
void Lacewing::Webserver::IdleTimeout (int)
C
long lw_ws_idle_timeout (lw_ws *)
C
void lw_ws_set_idle_timeout (lw_ws *, long)
JS
webserver.idleTimeout (<timeout>)

Description

Getter/setter for the default idle timeout in seconds (the time a connection can stay open without making a complete request).

A connection will never be closed when a request is in progress, even when the idle timeout is enabled. A partial request, however, does not count against the timeout.

The timeout can also be adjusted per Request with Request::IdleTimeout.
