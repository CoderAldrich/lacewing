SendConstant (Webserver::Request)

C++
void Lacewing::Webserver::Request::Send (const char * Data, int Size = -1)
C
void lw_ws_req_send_const (lw_ws_req *, const char * data, long size)
C
void lw_ws_req_send_text_const (lw_ws_req *, const char * text)
JS
The Webserver class is not currently supported in Javascript

Description

Sends data as part of the response to the Request, without using internal buffering.

As the data is not buffered internally, the application must be sure that the data will not change before the request is "finished". Lacewing does not usually guarantee when this will happen, so the data must outlive the Webserver itself.

However, if manual request finishing is enabled, the data need only remain valid until after Finish has returned.

Parameters

Data

A pointer to the data to send.

Size [optional]

The size of the data in bytes, or -1 if the data is a null-terminated string.
