Reset (Webserver::Request)

C++
void Lacewing::Webserver::Request::Reset ()
C
void lw_ws_req_reset (lw_ws_req *)
JS
request.reset ()

Description

Clears any response data added with Send, SendFile or SendConstant.

Anything else set as part of the response (eg. cookies) will be preserved.
