Hostname (Webserver::Request)

C++
const char * Lacewing::Webserver::Request::Hostname ()
C
const char * lw_ws_req_hostname (lw_ws_req *)
JS
request.hostname

Description

Returns the hostname, as provided by the HTTP "Host" header).

This is nearly the same as calling Header ("Host"), but Hostname removes any port number.
