SetAutoSave (Webserver::Upload)

C++
void Lacewing::Upload::SetAutoSave ()
C
const char * lw_ws_upload_set_auto_save (lw_ws_upload *)
JS
Function not currently supported

Description

Enables "auto save" mode for an Upload object.

When auto save mode is enabled, Lacewing will internally save the upload to a temporary file, of which the filename can be retrieved with GetAutoSaveFilename.

The onUploadChunk and onUploadDone handlers will not be called for a file when auto save mode is enabled - the next time the Upload object will be passed to the application is in onUploadPost itself.
