set_autosave (webserver::upload)

C++
void lacewing::webserver::upload::set_autosave ()
C
const char * lw_ws_upload_set_autosave (lw_ws_upload)

Description

Enables "auto save" mode for an active upload.

When auto save mode is enabled, the library will internally save the upload to a temporary file, of which the filename can be retrieved with autosave_filename.

The on_upload_chunk and on_upload_done hooks will not be called for a file when auto save mode is enabled - the next time the upload object will be passed to the application is in on_upload_post.
