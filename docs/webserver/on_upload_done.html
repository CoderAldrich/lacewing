on_upload_done (webserver)

C++
void on_upload_done (lacewing::webserver webserver, lacewing::webserver_request request,
                       lacewing::webserver_upload upload)
{
}

webserver->on_upload_done (on_upload_done); /* to register the hook */
C
void on_upload_done (lw_ws webserver, lw_ws_request, lw_ws_upload upload)
{
}

lw_ws_on_upload_done (webserver, on_upload_done); /* to register the hook */

Description

The on_upload_done hook is called to indicate that no more calls to on_upload_chunk will be made for this upload.

When all upload objects passed to on_upload_start for this request have also had on_upload_done called, the on_upload_post hook will be called to indicate a complete upload post request.

This hook will never be called if upload::set_autosave was called in on_upload_start.
