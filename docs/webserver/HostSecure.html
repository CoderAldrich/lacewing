HostSecure (Webserver)

C++
void Lacewing::Webserver::HostSecure (int Port = 443)
C++
void Lacewing::Webserver::HostSecure (Lacewing::Filter &Filter)
C
void lw_ws_host_secure (lw_ws *, long port)
C
void lw_ws_host_secure_filter (lw_ws *, lw_filter *)

Description

Start hosting the HTTPS server on the specified port, or with the specified Filter.

It is necessary to load a certificate using LoadCertificateFile or LoadSystemCertificate before calling this function.

Parameters

Port

The port to host on (default: 443)

OR Filter

The Filter to host with
