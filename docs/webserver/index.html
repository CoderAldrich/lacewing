Lacewing::Webserver

Description

Webserver aims to provide an embeddable, flexible HTTP server using liblacewing classes as a base.

Ease of embedding makes Webserver ideal for simpler things such as remote administration interfaces, but through the use of EventPump, Webserver is able to scale well and adapt transparently to different ways of handling I/O.

Features of Webserver include :-

  • Simple handlers for requests, with easy access to GET and POST data
  • Support for cookies and session data
  • Support for HTTPS (see HostSecure)
  • HTTP file upload

Hierachy

Webserver uses Server as a base for socket communication.
