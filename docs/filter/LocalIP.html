Function not currently supported

Description

Getter/setter for the local IP address associated with a Filter object, as a 32-bit integer.

This can be used to restrict a server-side class to listening on a particular IP address, when the system has more than one.

IPV6 - This function has been marked as incompatible with future IPV6 support, and should be avoided.
