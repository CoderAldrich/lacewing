filter.remote ([<Address or string>])

Description

Getter/setter for the remote address associated with a Filter object.

This can be used to restrict a server-side class to listening for data from a particular address.

Parameters

Address

An Address object.

OR Hostname

Any string an Address object can resolve. May also contain a port number (eg. "example.com:80").
