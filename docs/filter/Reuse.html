JS
filter.reuse ([<true or false>])

Description

Getter/setter for the SO_REUSEADDR option for this Filter.

This is set to false by default. When enabled, it allows a server-side class to bind to a port marked as "busy" (ie. potentially already bound by something else).

This option is often convenient when developing an application on Unix, as ports are not released immediately when a process exits.
