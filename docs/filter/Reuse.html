void lw_filter_reuse (lw_filter, lw_bool)

Description

Getter/setter for the SO_REUSEADDR option for this filter.

This is set to false by default. When enabled, it allows a server-side class to bind to a port marked as busy (i.e. potentially already bound by something else).

This option is often convenient when testing an application on operating systems where ports are not released immediately when a process exits.
