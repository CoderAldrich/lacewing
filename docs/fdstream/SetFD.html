SetFD (FDStream)

C
void lw_fdstream_setfd (lw_stream * fdstream, lw_fd FD, lw_bool auto_close)

Description

Sets the file descriptor for this FDStream.

Parameters

FD

The file descriptor (Unix) or HANDLE (Windows) to use.

auto_close [optional]

Set to true to close the file descriptor automatically when this FDStream is destroyed.
