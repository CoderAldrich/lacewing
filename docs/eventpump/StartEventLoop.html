StartEventLoop (EventPump)

C++
Lacewing::Error * Lacewing::EventPump::StartEventLoop ()
C
lw_error * lw_eventpump_start_event_loop (lw_pump * eventpump)

Description

Blocks until PostEventLoopExit is called, processing and dispatching events as necessary.

This method of using EventPump is particularly suited to dedicated console applications or daemons, which don't already have any other type of event loop.

Error Handling - As EventPump has no handlers, any error will be returned directly from this function as a pointer to an Error object. If no error occurred, 0 will be returned.

If an error does occur, it is the responsibility of the application to free the resulting Error object using delete (C++) or lw_error_delete (C).
