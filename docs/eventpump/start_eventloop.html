start_eventloop (eventpump)

C++
lacewing::error lacewing::eventpump::start_eventloop ()
C
lw_error lw_eventpump_start_eventloop (lw_eventpump)

Description

Blocks until post_eventloop_exit is called, processing and dispatching events as necessary.

This method of using eventpump is particularly suited to dedicated console applications or daemons, which don't already have any other type of event loop with which the library needs to integrate.

Error Handling - As eventpump has no hooks, any error will be returned directly from this function as a pointer to an error object. If no error occurred, 0 will be returned.

If an error does occur, it is the responsibility of the application to free the resulting error object using error_delete (C++) or lw_error_delete (C).
