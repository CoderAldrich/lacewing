tick (eventpump)

C++
lacewing::error lacewing::eventpump::tick ()
C
lw_error lw_eventpump_tick (lw_eventpump)

Description

Causes the eventpump to dispatch any currently pending events, then returns immediately.

If any hooks need to run, they are guaranteed to be called before tick returns. This function never blocks.

This method of using eventpump is particularly suited to games, which can simply place a call to tick in the gameloop to have all attached classes function properly.

Error Handling - As eventpump has no hooks, any error will be returned directly from this function as a pointer to an error object. If no error occurred, 0 will be returned.

If an error does occur, it is the responsibility of the application to free the resulting error object using error_delete (C++) or lw_error_delete (C).
