Tick (EventPump)

C++
Lacewing::Error * Lacewing::EventPump::Tick ()
C
lw_error * lw_eventpump_tick (lw_pump * eventpump)

Description

Causes the EventPump to dispatch any currently pending events, then returns immediately.

If any handlers need to run, they are guaranteed to trigger before Tick returns. This function never blocks.

This method of using EventPump is particularly suited to games, which can simply place a call to Tick in the gameloop to have all attached classes function properly.

Error Handling - As EventPump has no handlers, any error will be returned directly from this function as a pointer to an Error object. If no error occurred, 0 will be returned.

If an error does occur, it is the responsibility of the application to free the resulting Error object using delete (C++) or lw_error_delete (C).
