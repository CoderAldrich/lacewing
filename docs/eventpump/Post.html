Post (EventPump)

C++
void Lacewing::EventPump::Post (void * Function, void * Parameter)
C
void lw_eventpump_post (lw_eventpump *, void * function, void * parameter)
JS
eventPump.post (function)

Description

Posts a function to be called safely as part of the event loop.

The function specified by Function will be called with the single parameter specified by Parameter at a later time, in the same way that I/O readiness would be reported to liblacewing classes.

For example, if the application was a game calling Tick every frame, the function would be called the next time Tick was called. If a thread is currently blocking on StartEventLoop, that thread would call the function as soon as any existing ready I/O had finished propogating.

This function is thread-safe.

Parameters

Function

A pointer to a C-style function to call.

Parameter

A single pointer-sized parameter to pass to the function. If multiple parameters are required, consider passing a pointer to a struct.
