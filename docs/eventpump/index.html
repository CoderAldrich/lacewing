Lacewing::EventPump

Description

EventPump is a default implementation of Pump provided by the library, powered by IOCP, kqueue or epoll.

As there is no single model for event handling that would suit every application, EventPump provides several different modes of operation :-

  • Tick polls aggressively, suited to a game loop;

  • StartEventLoop starts a simple "block forever" event loop, suited to console applications and daemons;

  • StartSleepyTicking enables a threaded callback to request the application call Tick as soon as possible, suited to GUI applications with an existing message pump.
