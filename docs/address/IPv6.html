bool Lacewing::Address::IPv6 ()
C
lw_bool lw_addr_is_ipv6 (lw_addr *)

Description

Returns true if this is an IPv6 address, or false otherwise.

To create an IPv4 address, pass HINT_IPv4 to the Address constructor.
