IP_Byte (Address)

C++
int Lacewing::Address::IP_Byte (int Index)
C
long lw_addr_ip_byte (lw_addr *, long index)
JS
The Address class is not currently supported in Javascript

Description

Returns a single octet of the IP address.

IPV6 - This function has been marked as incompatible with future IPV6 support, and should be avoided.

Parameters

Index

The index of the octet to retrieve (0 - 3)
