resolve (address)

C++
lacewing::error lacewing::address::resolve ()
C
lw_error lw_addr_resolve (lw_addr)

Description

Blocks until the address has been resolved and is ready for use.

Returns 0 if the address was resolved successfully, or a pointer to an error object otherwise.
