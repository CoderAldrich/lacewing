Lacewing::SpinSync::WriteLock

Description

Create an instance of a WriteLock object to acquire exclusive write access to the resource protected by a SpinSync.

Only one WriteLock object may ever be active, and never at the same time as a ReadLock object.

If a ReadLock or another WriteLock is currently active, the WriteLock constructor will block.
