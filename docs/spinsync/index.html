Lacewing::SpinSync

Description

SpinSync provides a cross-platform synchronization object with a "multiple readers, one writer" model.

It enables the protection of sections of code or variables from simultaneous access by multiple writers, while allowing multiple readers when no writer is active.

For a simpler synchronization object where all lockers are treated as potential writers, see Sync.
