on_tick (timer)

C++
void on_tick (lacewing::timer timer)
{
}

timer->on_tick (on_tick); /* to register the hook */
C
void on_tick (lw_timer timer)
{
}

lw_timer_ontick (timer, on_tick); /* to register the hook */

Description

This hook triggers once for every tick of the timer, at the interval previously specified as the parameter to start.
