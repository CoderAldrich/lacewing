lacewing::timer

Description

The timer class provides a simple eventpump driven timer, useful for implementing a "heartbeat" in server applications.

timer is implemented using Timer FDs on Linux, kqueue's EVFILT_FILTER, or a thread and #eventpump::post# on other systems.
