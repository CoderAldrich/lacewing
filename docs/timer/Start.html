Start (Timer)

C++
void Lacewing::Timer::Start (int Milliseconds)
C
void lw_timer_start (lw_timer *, long milliseconds)

Description

Starts the Timer, causing the onTick handler to be called at the specified interval.

If the Timer is already started, it will restart with the new interval.

Parameters

Milliseconds

The interval to call the onTick handler, in milliseconds.
