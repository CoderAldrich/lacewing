start (timer)

C++
void lacewing::timer::start (int msec)
C
void lw_timer_start (lw_timer, long msec)

Description

Starts the timer, causing the on_tick hook to be called at the specified interval.

If the timer is already started, it will restart with the new interval.

Parameters

msec

The interval to call the on_tick hook, in milliseconds.
