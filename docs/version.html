version

C++
const char * version ()
C
const char * lw_version ()

Description

Returns a string identifying the current version of liblacewing.

The format of this string is not guaranteed, but it usually contains the last library release number and the operating system type, for example:

    liblacewing 0.2.3 (FreeBSD/x64)

This function is used by webserver to generate the default HTTP Server header.
