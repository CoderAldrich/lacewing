load_sys_cert (server)

C++
bool lacewing::server::load_sys_cert (const char * store_name,
         const char * common_name, const char * location = "currentuser")
C
lw_bool lw_server_load_sys_cert (lw_server, const char * store_name,
                 const char * common_name, const char * location)

Description

Loads a certificate from the system. This function is only available on Windows.

Returns true if the certificate was loaded successfully, or false otherwise.

Parameters

store_name

The name of the store to open, as described in System Store Locations (Windows).

Defaults to "MY" if left blank.

common_name

A common name matching the certificate to load. This is usually the name of the server, such as "localhost" or "example.com".

Location [optional]

The system location in which the certificate is stored.

This is a string containing one of the following (a blank string will default to "CurrentUser"):

• CurrentUser
• CurrentUserGroupPolicy
• LocalMachine
• LocalMachineEnterprise
• LocalMachineGroupPolicy
• Services
• Users
