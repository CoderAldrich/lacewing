on_data (server)

C++
void on_data (lacewing::server server, lacewing::server::client client, char * data, size_t size)
{
}

server->on_data (on_data); /* to register the hook */
C
void on_data (lw_server server, lw_client client, char * data, size_t size)
{
}

lw_server_on_data (server, on_data); /* to register the hook */

Description

The on_data hook is called when data is received from a client.

on_connect will always be called before this hook, and this hook will never be called after on_disconnect.
