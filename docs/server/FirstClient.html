FirstClient (Server)

C++
Lacewing::Server::Client * Lacewing::Server::FirstClient ()
C
lw_stream * lw_server_first_client (lw_server *)

Description

Returns a pointer to the first Client on the Server, or 0 if no clients are connected.

This can be used to iterate through all clients using Client::Next
