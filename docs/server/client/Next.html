Next (Server::Client)

C++
Lacewing::Server::Client * Lacewing::Server::Client::Next ()
C
lw_stream * lw_server_client_next (lw_stream * client)

Description

Returns the next Client on the Server, or 0 if this Client is the last.

Combined with Server::FirstClient, this can be used to iterate through the entire client list. The order of the clients is unspecified.
