next (server::client)

C++
lacewing::server_client lacewing::server_client::next ()
C
lw_stream lw_server_client_next (lw_server_client)

Description

Returns the next client on the server, or 0 if this client is the last.

Combined with server::client_first, this can be used to iterate through the entire client list. The order of the clients is unspecified.
