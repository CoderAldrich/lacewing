Disconnect (Server::Client)

C++
void Lacewing::Server::Client::Disconnect ()
C
void lw_server_client_disconnect (lw_server_client *)
JS
The Server class is not currently supported in Javascript

Description

Forcefully disconnects the Client. The onDisconnect handler will be called soon afterwards.

Until onDisconnect is called, the Client object remains valid to use.
