SendFile (Server::Client)

C++
void Lacewing::Server::Client::SendFile (const char * Filename, lw_i64 Offset = 0, lw_i64 Size = -1)
C
void lw_server_client_sendfile (lw_server_client *, const char * filename)

void lw_server_client_sendfile_ex (lw_server_client *, const char * filename, lw_i64 offset, lw_i64 size)
JS
The Server class is not currently supported in Javascript

Description

Sends data from a file to the Client, optionally with an offset and size.

This function is non-blocking, and the order of the file data in relation to any other calls to Send or SendFile is guaranteed to be the same as the order the functions were called in.

Internally, SendFile will usually result in calls to a kernel function such as sendfile on Unix, or TransmitFile on Windows. Where this is not possible (ie. with secure sockets), the file will be sent in chunks.

Parameters

Filename

The filename of the file to send.

Offset [optional]

The offset in the file to start sending from, in bytes.

Size [optional]

The number of bytes to send, or -1 to send to the end of the file.
