SendWritable (Server::Client)

C++
void Lacewing::Server::Client::SendWritable (char * Data, int Size = -1)
C
void lw_server_client_send_writable (lw_server_client *, char * data, long size)
JS
The Server class is not currently supported in Javascript

Description

Sends some data to the Client, but encrypts the data in place when in secure mode.

Although this avoids a copy and is therefore likely faster than Send, it means the buffer will contain garbage after SendWritable returns.

Parameters

Data

A pointer to the data to send. The data must be writable.

Size [optional]

The size of the data in bytes, or -1 if the data is a null-terminated string.
