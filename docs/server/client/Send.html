Send (Server::Client)

C++
void Lacewing::Server::Client::Send (const char * Data, int Size = -1)
C
void lw_server_client_send (lw_server_client *, const char * data, long size)

C
void lw_server_client_sendf (lw_server_client *, const char * format, ...)
JS
The Server class is not currently supported in Javascript

Description

Sends some data to the Client. For convenience, this is also available as the << operator in C++.

Parameters

Data

A pointer to the data to send.

Size [optional]

The size of the data in bytes, or -1 if the data is a null-terminated string.
