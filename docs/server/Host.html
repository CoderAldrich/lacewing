Description

Start hosting the server on the specified port, or with the specified Filter.

If a certificate has previously been loaded successfully (that is, CertificateLoaded returns true), the Server will host in secure mode.

Parameters

Port

The port to host on

OR Filter

The Filter to host with

ClientSpeaksFirst [optional]

Pass true if the protocol you are working with has the client send data to the server first (eg. HTTP). This will enable an optimisation in which data is received as part of the AcceptEx call on Windows.
