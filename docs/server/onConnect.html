onConnect (Server)

C++
void onConnect (Lacewing::Server &Server, Lacewing::Server::Client &Client)
{
}

Server.onConnect (onConnect); /* to register the handler */
C
void on_connect (lw_server * server, lw_server_client * client)
{
}

lw_server_onconnect (server, on_connect); /* to register the handler */
JS
The Server class is not currently supported in Javascript

Description

The onConnect handler is called when a Client connects to the Server.

The Client object will be valid until the onDisconnect handler is called. This handler will always be called before onReceive or onDisconnect.
