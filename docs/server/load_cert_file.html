load_cert_file (server)

C++
bool lacewing::server::load_cert_file (const char * filename, const char * passphrase = "")
C
lw_bool lw_server_load_cert_file (lw_server, const char * filename, const char * passphrase)

Description

Loads a certificate from a file. The file may be a PEM file (Unix) or a CER/PFX file (Windows Vista or greater).

Returns true if the certificate was loaded successfully, or false otherwise.

This function is not supported on versions of Windows prior to Windows Vista.
