can_npn (server)

C++
bool lacewing::server::can_npn ()
C
lw_bool lw_server_can_npn (lw_server *)

Description

Returns true if TLS NPN (next protocol negotation) is supported by this server instance, false otherwise.

NPN is supported when liblacewing was compiled with a recent version of OpenSSL.
