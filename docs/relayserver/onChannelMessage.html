    bool Blasted, int Subchannel, char * Data, int Size, int Variant)

    lw_bool blasted, long subchannel, char * data, long size, long variant)
{
        return lw_true;

JS
The RelayServer class is not currently supported in Javascript

Description

The onChannelMessage handler is called when a channel message is received from a Client.

onJoinChannel will always be called before this handler, and this handler will never be called after a successful onLeaveChannel.

Return true to have the message relay to the rest of the channel as normal, or false to drop the message.
