ChannelMaster (RelayServer::Channel)

C++
Lacewing::RelayServer::Client * Lacewing::RelayServer::Channel::ChannelMaster ()
C
lw_relaysrv_client * lw_relaysrv_channel_channelmaster (lw_relaysrv_channel *)
JS
The RelayServer class is not currently supported in Javascript

Description

Returns a pointer to the Client marked as the channel master for this Channel.

If there is no channel master (ie. the channel master has already left), 0 will be returned.
