Send (RelayServer::Channel)

C++
void Lacewing::RelayServer::Channel::Send
    (int Subchannel, const char * Data, int Size = -1, int Variant = 0)
C
void lw_relaysrv_channel_send
    (lw_relaysrv_channel *, long subchannel, const char * data, long size, long variant)
JS
The RelayServer class is not currently supported in Javascript

Description

Sends data to the channel, with a subchannel and optional variant identifier.

Parameters

Subchannel

The subchannel to send the data on (0 - 255)

Data

A pointer to the data to send

Size [optional]

The size of the data in bytes, or -1 if the data is a null-terminated string

Variant [optional]

A user-defined integer ranging from 0 - 15 to identify the type of the data.
