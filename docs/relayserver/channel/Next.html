Next (RelayServer::Channel)

C++
Lacewing::RelayServer::Channel * Lacewing::RelayServer::Channel::Next ()
C
lw_relaysrv_channel * lw_relaysrv_channel_next (lw_relaysrv_channel *)
JS
The RelayServer class is not currently supported in Javascript

Description

Returns the next Channel on the RelayServer, or 0 if this Channel is the last.

Combined with RelayServer::FirstChannel, this can be used to iterate through the entire channel list. The order of the channels is unspecified.
