onConnect (RelayServer)

C++
void onConnect (Lacewing::RelayServer &Server, Lacewing::RelayServer::Client &Client)
{
}

Server.onConnect (onConnect); /* to register the handler */
C
void on_connect (lw_relaysrv * server, lw_relaysrv_client * client)
{
}

lw_relaysrv_onconnect (server, on_connect); /* to register the handler */
JS
The RelayServer class is not currently supported in Javascript

Description

The onConnect handler is called when a Client connects to the RelayServer.

The Client object will be valid until the onDisconnect handler is called. This handler will always be called before any other handlers a Client object is passed to.
