onDisconnect (RelayServer)

C++
void onDisconnect (Lacewing::RelayServer &Server, Lacewing::RelayServer::Client &Client)
{
}

Server.onDisconnect (onDisconnect); /* to register the handler */
C
void on_disconnect (lw_relaysrv * server, lw_relaysrv_client * client)
{
}

lw_relaysrv_ondisconnect (server, on_disconnect); /* to register the handler */
JS
The RelayServer class is not currently supported in Javascript

Description

The onDisconnect handler is called when a Client disconnects from the RelayServer.

After this handler has returned, the Client object is no longer valid. onConnect will always be called before this handler.
