RemoveUser (Pump)

C++
void Lacewing::Pump::RemoveUser ()
C
void lw_pump_remove_user (lw_pump *)

Description

Decrements the number of active users of this Pump. When no users are left, InUse will return false.

This will be called by each class when they are finished using the Pump.
