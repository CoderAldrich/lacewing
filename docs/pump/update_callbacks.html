update_callbacks (pump)

C++
void lacewing::pump::update_callbacks (pump::watch, void * tag, <callback>)
C++
void lacewing::pump::update_callbacks
    (pump::watch, void * tag, <read_callback>, <write_callback>, bool edge_triggered = true)
C
void lw_pump_update_callbacks (pump::watch, void * tag, <callback>)
C
void lw_pump_update_callbacks
    (pump::watch, void * tag, <read_callback>, <write_callback>, lw_bool edge_triggered)

Description

Changes the tag and callbacks for an active watch with immediate effect.

Instead of a file descriptor, the first parameter is a pointer to an active watch. Otherwise, the semantics are exactly the same as add.
