post (pump)

C++
void lacewing::pump::post (void * function, void * parameter)
C
void lw_pump_post (lw_pump *, void * function, void * parameter)

Description

Posts a function to be called safely as part of the event loop.

The function specified by function will be called with the single pointer-sized parameter specified by parameter in the same way that a pump callback would be executed.

For example, if the application was a game calling eventpump::tick every frame, the function would be called the next time eventpump::tick was called. If a thread is currently blocking on eventpump::start_eventloop, that thread would call the function as soon any pending operations had completed.

This function is thread-safe.

Parameters

function

A pointer to a C-style function to call.

parameter

A single pointer-sized parameter to pass to the function. If multiple parameters are required, a pointer to a structure may be passed.
