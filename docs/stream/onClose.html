onClose (Stream)

C++
void onClose (Lacewing::Stream &)
{
}

Stream.AddHandlerClose (onClose); /* to register the handler */
C
void on_close (lw_stream * stream)
{
}

lw_stream_add_handler_close (stream, on_close); /* to register the handler */

Description

The onClose handler is triggered when the Stream is closing, either through being deleted or an explicit call to Close.

Unlike most handlers in the library, a stream may have more than one onClose handler (the most recently added will be called first.)

AddHandlerClose also allows a pointer-sized tag to be specified, which will be included when the handler is called.
