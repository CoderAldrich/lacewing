EndQueue (Stream)

C++
void Lacewing::Stream::EndQueue ()
C++
void Lacewing::Stream::EndQueue (int head_buffers, const char ** buffers, size_t * lengths)
C
void lw_stream_end_queue (lw_stream *, int head_buffers, const char ** buffers, size_t * lengths)

Description

Stops queueing, flushing the queue upstream.

Optionally, a set of head buffers may be passed to be written before the queue. This is useful for e.g. including HTTP headers before the (already buffered) response body.

Parameters

head_buffers

The number of head buffers to write before the queue.

buffers

An array of pointers to the head buffers.

lengths

An array of lengths for the head buffers (a length of -1 indicates a null-terminated string.)
