WriteFile (Stream)

C++
void Lacewing::Stream::WriteFile (const char * filename)
C
void lw_stream_write_file (lw_stream *, const char * filename)

Description

Writes a file to this Stream by filename.

This is equivalent to creating a new File and splicing it using Write.

Parameters

filename

The filename of the file to write.
