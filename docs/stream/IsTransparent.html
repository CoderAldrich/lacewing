IsTransparent (Stream)

C++
bool Lacewing::Stream::IsTransparent ()
C
lw_bool lw_stream_is_transparent (lw_stream *)

Description

Returns true if this Stream should be considered transparent (i.e. any upstream data will simply become downstream data without side-effects.)

This is called by the library when making decisions on how to splice streams. If, for example, a #Pipe# is between two FDStreams and has no data handlers registered, kernel sendfile or TransmitFile may still be used, ignoring the transparent intermediate.
