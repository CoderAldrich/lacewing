on_data (stream)

C++
void on_data (lacewing::stream, void * tag, const char * buffer, size_t size)
{
}

stream->add_hook_data (on_data); /* to register the hook */
C
void on_data (lw_stream, void * tag, const char * buffer, size_t size)
{
}

lw_stream_add_hook_data (stream, on_data); /* to register the hook */

Description

The on_data hook is triggered when the stream has downstream data to emit.

Unlike most hooks in the library, a stream may have more than one on_data hook (the most recently added will be called first.)

add_hook_data also allows a pointer-sized tag to be specified, which will be included when the hook is called.
