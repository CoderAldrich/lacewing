Read (Stream)

C++
void Lacewing::Stream::Read (size_t bytes)
C
void lw_stream_read (lw_stream *, size_t bytes)

Description

Tries to read downstream data from the Stream. When data has been read successfully, the onData handlers will be invoked.

Note that Read is just a hint; the actual amount of data may vary, and streams are permitted to emit data even when Read has not been called.

Parameters

bytes

The number of bytes to read, or -1 to read everything the Stream can produce.
