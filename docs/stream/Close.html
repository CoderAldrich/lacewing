close (stream)

C++
bool lacewing::stream::close (bool immediate)
C
lw_bool lw_stream_close (lw_stream, lw_bool immediate)

Description

By default, cancels all I/O operations and calls any close hooks.

Most subclasses override close to do something more specific (e.g. disconnect the socket, close the file handle, ...)

Parameters

immediate

If immediate is true and data is still waiting to be written, the stream will be closed when the write is complete (on_close may be used to determine when this is). The return value will indicate whether the stream was closed immediately or not.

If immediate is false and data is still waiting to be written, the stream will be closed immediately and close will always return true. This may result in data being truncated.
