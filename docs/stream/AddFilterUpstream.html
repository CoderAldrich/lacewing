AddFilterUpstream (Stream)

C++
void Lacewing::Stream::AddFilterUpstream (bool delete_with_stream = false)
C
void lw_stream_add_filter_upstream (lw_stream *, lw_stream * filter, lw_bool delete_with_stream)

Description

Adds a stream as a filter for upstream data.

Any data written to this stream will pass through the filter stream first, starting with the most recently added.

Filters can be used, for example, to transparently add a layer for things like SSL or compression to an existing stream.

Parameters

delete_with_stream [optional]

If true, the filter will automatically be deleted with this stream.
