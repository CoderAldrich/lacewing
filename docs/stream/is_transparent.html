is_transparent (stream)

C++
bool lacewing::stream::is_transparent ()
C
lw_bool lw_stream_is_transparent (lw_stream)

Description

Returns true if this stream should be considered transparent (i.e. any upstream data will simply become downstream data without side-effects.)

This is called by the library when making decisions on how to splice streams. If, for example, a pipe is between two fdstreams and has no data hooks registered, kernel sendfile or TransmitFile may still be used, ignoring the transparent intermediate stream.
