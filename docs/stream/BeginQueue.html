BeginQueue (Stream)

C++
void Lacewing::Stream::BeginQueue ()
C
void lw_stream_begin_queue (lw_stream *)

Description

Starts queueing any written data or streams until EndQueue is called.

The number of queued bytes may be retrieved with Queued.
