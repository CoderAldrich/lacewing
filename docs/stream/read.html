read (stream)

C++
void lacewing::stream::read (size_t bytes)
C
void lw_stream_read (lw_stream, size_t bytes)

Description

Tries to read downstream data from the stream. When data has been read successfully, any registered on_data hooks will be called.

Note that read is just a hint; the actual amount of data produced is not strictly controlled, and streams may emit data even when read has not been called.

Parameters

bytes

The number of bytes to read, or -1 to read everything the stream can produce.
