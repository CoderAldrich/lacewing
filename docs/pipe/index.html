    • Lacewing::Pipe

    Description

    Pipe is a simple Stream with which any upstream data becomes downstream data without modification.

    The library is aware of this through Stream::IsTransparent, so (providing there are no data handlers), a Pipe as an intermediate between two other streams will be ignored when deciding on the optimal way to transfer data.
