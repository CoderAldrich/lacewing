What's all this Javascript stuff about?

To cut a long story short, the perpetrator of this madness is a fan of Javascript as a language, and thought it would be a good idea to wrap the liblacewing library.

The Javascript bindings are still experimental, and only Webserver is supported for the time being. They target SpiderMonkey and V8, and liblacewing can also be compiled as an add-on for Node.JS.

They're probably a bit early in development to use for anything serious, but feel free to try and build if you're interested, and report back with what's broken.
