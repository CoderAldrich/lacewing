Clone (Error)

C++
Lacewing::Error * Lacewing::Error::Clone ()
C
lw_error * lw_error_clone (lw_error *)

Description

Creates a clone of an Error object, preserving all error information.

The caller must remember to free the clone using delete (C++) or lw_error_delete (C).
