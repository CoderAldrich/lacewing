ToString (Error)

C++
const char * Lacewing::Error::ToString ()
C++
Lacewing::Error::operator const char * ()
C
const char * lw_error_tostring (lw_error *)
JS
error.toString ()

Description

Returns a formatted error message for an Error object.

The string will be valid as long as the Error object itself is in scope (this does not apply in Javascript). If more errors are added using Add, ToString should be called again and the old string considered invalid.
