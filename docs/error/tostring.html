tostring (error)

C++
const char * lacewing::error::tostring ()
C++
lacewing::error::operator const char * ()
C
const char * lw_error_tostring (lw_error)

Description

Returns a formatted error message for an error object.

The string will be valid as long as the error object is in scope. If more errors are added using add, tostring should be called again and the old string considered invalid.
