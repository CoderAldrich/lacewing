Description

Adds an error as the highest level in an error object.

For example, an error for a failed system call will initially contain the error text provided by the system.

Text describing the type of system call that failed (eg. "Error receiving") will then be added, and an even higher level class may add something like "Socket error" before reporting the error to the application.

Parameters

format

The format of the error string, followed by a dynamic argument list (similar to the C function printf.

OR error_code

A numeric error code reported by a system call. Internally, this will be converted to a string by FormatMessage on Windows, or strerror on Unix.
