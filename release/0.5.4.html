Release Notes - 0.5.4

The 0.5.4 release of liblacewing is now available.

Changes in this version

  • The post_eventloop_exit function is now implemented on UNIX platforms.
  • The socket address information is now stored and retrievable for server and webserver clients. This fixes functions such as lw_ws_req_addr, which previously returned NULL (a 0.5.x regression.)
  • Suppressed warnings about unused lwp_release function when compiling the library
  • Fix issue with the webserver accessing freed pointers when writing response headers. This could cause segmentation faults or heap corruption in some cases.
