Release Notes - 0.5.3

The 0.5.3 release of liblacewing is now available.

This release implements start_sleepy_ticking on UNIX, which allows the library to be used easily from applications with an existing event loop.

An example of a fully cross-platform Qt application using liblacewing can be found here.

Other changes

  • lw_event now supports infinite timeouts when waiting for events
  • The C++ wrapper for pump:;add now returns the lw_pump_watch
  • The makefile now respects autoconf libdir and includedir. Previously, the library would always install to $(PREFIX)/lib, which was incorrect on (for example) 64-bit Fedora systems where lib64 is used instead.
  • The makefile now produces a dynamic library without the version suffix (i.e. liblacewing.so instead of liblacewing.so.0.5.3). make install will still install the library with the version suffix.
